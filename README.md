# Non-Functional Four-Function Calculator

This is an intentionally "non-functional" Python calculator. It was created for **educational purposes** to demonstrate how to build logic entirely through control flow (like `if/elif/else` and loops) *without* defining any custom Python `def` functions.

## 🚀 Concept
By intentionally avoiding functions, this project serves as a practice exercise in mastering raw control structures and state management in a procedural, top-to-bottom execution flow.

## ⚙️ Usage
Simply run the script:
```bash
python calculator.py
```
